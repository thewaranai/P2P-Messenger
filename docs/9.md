# Диаграмма последовательности (отправка сообщения)

Сценарий: **Пользователь A отправляет сообщение пользователю B**

```mermaid
sequenceDiagram
    actor UserA as Пользователь A
    actor UserB as Пользователь B
    participant UI_A as ChatActivity (A)
    participant CM_A as ConnectionManager (A)
    participant CM_B as ConnectionManager (B)
    participant DB_A as Database (A)
    participant DB_B as Database (B)

    UserA->>UI_A: вводит текст и нажимает "Отправить"
    UI_A->>CM_A: sendMessage(text)
    CM_A->>CM_A: создаёт объект Message
    CM_A->>DB_A: сохранить сообщение (локально)
    CM_A->>CM_B: отправляет данные по P2P-сокету
    CM_B->>DB_B: сохранить полученное сообщение
    CM_B->>UI_B: уведомление о новом сообщении
    UI_B->>UserB: отобразить сообщение
    CM_B-->>CM_A: подтверждение доставки
    CM_A->>UI_A: обновить статус сообщения (доставлено)
